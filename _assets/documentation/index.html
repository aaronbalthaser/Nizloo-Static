Buttons

Buttons are creating using the button mixins located in _partials -> mixins -> _buttons. The file contains the following mixins:

  • button()
  • button-size()
  • button-fluid()
  1. button(): Defines the button theme properties including text color, background color and border color.
  2. button-size(): Defines button dimensions including vertical & horizontal padding, font size, line height and border radius.
  3. button-fluid(): Includes CSS properties forcing the button to span as wide as it's container.

Add the mixins to a class or id.

Default Button Properties

Default properties can be changed in the _variables mixin file, however it is not necessary to do so. Default properties protect against errors during compilation if parameters are incorrect or missing.

Overriding Default Properties

Default button properties can be overridden by setting default values in the Variables file or by passing the values as parameters into the mixin.

  • Variables File
    • Go to variables.scss and add default values to buttons theme.
  • Mixin Parameters
    • Theme
      • $btn-color: #eeeeee
      • $btn-background: #eeeeee
      • $btn-border: #eeeeee
    • Dimensions
      • $btn-padding-vertical: 8px
      • $btn-padding-horizontal: 120px
      • $btn-font-size: 18px
      • $btn-line-height: 1.7
      • $btn-border-radius: none
